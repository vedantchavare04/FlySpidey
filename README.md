**FlySpidey - A Python Game using Pygame**<br/>
**Overview-**

FlySpidey is a fun Python game developed using the Pygame library. The objective of the game and core gameplay mechanics can be customized based on your design. The game is built and executed in PyCharm, with the main logic running through the main.py file.

**Requirements-**

Make sure you have the following installed:

Python 3.x

Pygame library

PyCharm IDE (optional but recommended)

Installing Pygame

Run the following command in your terminal or PyCharm terminal:

pip install pygame

**How to Run the Game**

Clone or download this project repository.

Open the project in PyCharm or any code editor.

Ensure the main.py file is selected.

Run the script:

Using terminal:

python main.py

Or click the Run button in PyCharm.

**Game Controls-**

Click up arrow to fly spidey upwards

**Game Features-**

Tackle green goblin using one upward control


**Feel free to modify the game elements:**

Update images/sounds in the assets folder

Change gameplay mechanics in main.py
